# Data Structures and Algorithms 102

This repository covers the **Time and Space Complexity** module of the DSA 102 course. It introduces Big O notation and walks through how to calculate and improve both the time and space complexity of an algorithm.

## What you'll learn

- What time complexity and space complexity are, and why they matter when comparing algorithms
- How to read and reason about Big O notation (O(1), O(log n), O(n), O(n log n), O(n²), O(2ⁿ))
- Step-by-step rules for calculating an algorithm's time complexity
- Step-by-step rules for calculating an algorithm's space (auxiliary memory) complexity
